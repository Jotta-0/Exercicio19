**Autor:** João  
**Data:** 20 de fevereiro  

---

## Descrição

Este projeto consiste em um **formulário de cadastro de paciente** feito em **HTML e CSS simples**.  
O formulário coleta informações como nome, CPF, endereço, telefone, e-mail, data de nascimento, login e senha.  

Ele possui **validação básica** (campos obrigatórios), **botões para enviar e limpar** os dados, e **alerta de confirmação** quando o formulário é submetido.  
O objetivo é praticar a criação de formulários acessíveis, organizados e visualmente estilizados.
